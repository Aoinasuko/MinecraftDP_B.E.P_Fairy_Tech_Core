＝＝＝＝＝＝＝B.E.P Fairy Tech Core＝＝＝＝＝＝＝
青井ナスコ製作のデータパックの前提MODです。
専用のクラフティングテーブルや、実績用のデータを追加します。

＝＝＝＝＝＝＝追加アイテム＝＝＝＝＝＝＝
・*LittleFairy Crafter*
各種追加のデータパックで追加されるちょっと便利なアイテムを作成するために必要になります。
-作成方法-
周りに空間がある場所にディスペンサーを置き、その上に額縁を設置し
羽を額縁に入れることで、ディスペンサーが*LittleFairy Crafter*に変化します。
-注意-
作成時にディスペンサーの中身のアイテムは全て消滅します。

・*B.E.P Fairy Tech Crafter*
各種追加のデータパックで追加されるとても便利なアイテムを作成するために必要になります。
-作成方法-
周りに空間がある場所にシュルカーボックスを置き、その上に額縁を設置し
クラフティングテーブルを額縁に入れることで、シュルカーボックスが*B.E.P Fairy Tech Crafter*に変化します。
-注意-
作成時及び破壊時にシュルカーボックスの中身のアイテムは全て消滅します。
